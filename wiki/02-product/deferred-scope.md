# Deferred Scope

Deferred scope is not rejected. It is intentionally postponed to protect MVP focus.

## Properties deferred

- hard delete
- archive/inactive flow
- property images/files
- search/filter/pagination
- advanced address/location fields
- import/export

## Units deferred

- standalone Unit detail page
- status/occupancy workflow
- base_rent_amount/rent pricing workflow
- delete/archive/status-management
- top-level Units navigation
- images/files
- search/filter/pagination
- import/export

## Tenant deferred

- tenant portal/login
- identity document files
- emergency contacts
- tenant-to-unit direct assignment
- delete/archive flow

## Lease deferred

- renewal automation
- cancellation workflows
- generated contracts
- document signing
- advanced lease terms

## Billing and payments deferred

- automatic recurring billing
- late fees
- payment gateway integration
- advanced accounting
- tax handling

## Reminder deferred

- automated sending
- bulk campaigns
- template analytics
