# Product Overview

RuangRapi is a rental operations SaaS for Indonesian rental property owners.

The product helps owners manage the core rental workflow:

```txt
Properties
→ Units
→ Tenants
→ Leases
→ Billing / Invoices
→ Payments
→ Receipts
→ Reminders
→ Maintenance
→ Reporting
```
