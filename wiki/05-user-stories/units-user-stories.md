# Units User Stories

## Built

### View units under a property

As a rental owner, I want to see units under a property so that I understand what rentable spaces exist.

### Create unit

As a rental owner, I want to add a unit under a property so that I can later assign leases and tenants.

### Edit unit

As a rental owner, I want to edit unit name, type, and notes so that unit records stay accurate.

## Deferred stories

### Track unit occupancy

As a rental owner, I want to know whether a unit is vacant or occupied so that I can manage availability.

### Track unit base rent

As a rental owner, I want to store base rent for a unit so that billing can start from accurate rent terms.
