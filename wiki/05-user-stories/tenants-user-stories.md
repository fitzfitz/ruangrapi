# Tenants User Stories

## Not implemented

### List tenants

As a rental owner, I want to see my tenant records so that I can manage renters in one place.

### Create tenant

As a rental owner, I want to add a tenant contact so that I can later create a lease for them.

### Edit tenant

As a rental owner, I want to update tenant information so that contact records stay accurate.

## Deferred

### Attach tenant to unit

This should be handled through Leases later, not direct tenant-to-unit assignment in the first Tenants slice.

### Tenant portal

Deferred until core owner workflows are stable.
