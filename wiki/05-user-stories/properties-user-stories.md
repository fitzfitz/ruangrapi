# Properties User Stories

## Built

### List properties

As a rental owner, I want to see my properties so that I know what assets I manage.

### Create property

As a rental owner, I want to add a property so that I can start organizing units under it.

### View property detail

As a rental owner, I want to view a property detail page so that I can see its basic information and related units.

### Edit property

As a rental owner, I want to edit property information so that my records stay accurate.

## Deferred stories

### Archive property

As a rental owner, I want to archive inactive properties so that old records do not clutter my active operations while history remains preserved.
