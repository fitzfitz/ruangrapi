# Leases User Stories

## Not implemented

### Create lease

As a rental owner, I want to create a lease that connects a Tenant to a Unit so that I can track who rents what.

### View active leases

As a rental owner, I want to view active leases so that I know current rental agreements.

### End lease

As a rental owner, I want to end a lease so that historical agreements are preserved and the unit can become available later.

## Dependencies

Leases should come after Tenants.
