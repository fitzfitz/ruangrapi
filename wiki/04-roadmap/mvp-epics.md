# MVP Epics

## Epic 1: Foundation

Status: built

Goal: establish app, auth, onboarding, organization context, route gating, and dashboard shell.

## Epic 2: Properties

Status: built

Goal: manage top-level rental assets.

Built:

- list
- create
- detail
- edit

## Epic 3: Units

Status: built

Goal: manage rentable spaces under Properties.

Built:

- read-only Units section
- create Unit
- edit Unit

## Epic 4: Tenants

Status: built

Goal: manage renter/contact records.

Built:

- plan Tenants module
- read-only Tenants list
- create Tenant flow
- edit Tenant flow
- validation checklist
- Tenants module closeout

Deferred:

- lease assignment
- direct tenant-to-unit assignment
- delete/archive
- tenant portal
- document upload

## Epic 5: Leases

Status: built

Goal: connect Tenants to Properties/Units through rental agreements.

Built:

- plan Leases module
- read-only Leases list
- create Lease flow
- validation checklist
- Leases module closeout

Deferred:

- lease edit
- end lease
- cancel lease
- occupancy/status synchronization
- invoice generation
- deposit ledger
- contract files and document uploads

## Epic 6: Billing and Payments

Status: built

Goal: track what is due and what has been paid.

Built:

- plan Billing / Invoices module
- read-only Invoices list
- create draft rent Invoice flow
- minimal Invoice issue action
- Billing / Invoices validation checklist
- Billing / Invoices module closeout
- plan Payments module
- read-only Payments list
- record Payment flow
- Payments validation checklist
- Payments implementation closeout
- plan Receipts module
- Generate Receipt after Payment
- Receipts validation checklist

Deferred:

- invoice detail
- invoice edit
- invoice send/delivery
- invoice cancel
- receipt detail page
- receipt list page
- print/download receipt
- PDF generation
- email or WhatsApp delivery
- automatic receipt generation after payment recording
- payment edit blocking after receipt generation
- payment correction workflow
- reminders and WhatsApp messages
- automatic overdue status jobs
- utility reading capture and utility billing
- invoice PDFs, downloads, email, and WhatsApp delivery
- dashboard metrics

## Epic 7: Receipts and Reminders

Status: Receipts manual generation baseline complete; Reminders planning next after manual Receipts validation

Goal: support proof of payment and follow-up workflows.

## Epic 8: Maintenance and Reporting

Status: not started

Goal: support operational issue tracking and business visibility.
